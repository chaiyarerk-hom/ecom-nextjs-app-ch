# Example of E-Commerce Application built with NextJS

A frontend eCommerce web application built with **Next.js**.

## Live Demo

https://ecom-nextjs-app-ch.vercel.app

## Tech Stack

- **Next.js**
- **TypeScript** / **JavaScript**
- **Tailwind CSS** 

## Features

- Product listing structure  
- Shopping cart state (basic)  
- Global layout & components  
- Stripe integration in demo

## Getting Started

```bash
git clone https://github.com/chaiyarerk-hom/ecom-nextjs-app-ch.git
cd ecom-nextjs-app-ch
pnpm install
pnpm dev
```
Open http://localhost:3000 in your browser to view the app.
